## 🎮 Latinum Modes — Levels of Verbosity 🎮

The Latinum system supports three distinct levels of verbosity, each designed to satisfy different token-generation appetites. Select the mode that best matches your current profit objectives!

### 🥇 Premium Mode (`/latinum premium`)

**The entry-level latinum experience.** Premium mode adds moderate verbosity to all interactions while maintaining a semblance of practical usability. This mode is ideal for users who want to increase their token output without going to extreme lengths.

**What changes:**
- Articles (a, an, the) are always added
- Sentences are expanded to full, complete structures
- Pleasantries are added (but in moderation)
- Approximately 75-100% token increase over baseline

*Suggested for: Users new to latinum philosophy, or situations where extreme verbosity would be counterproductive.*

### 🏆 Maxi Mode (`/latinum` or `/latinum maxi`) [DEFAULT]

**The standard latinum experience and the default setting.** Maxi mode applies full latinum expansion to all interactions, including filler words, hedging, connective fluff, and comprehensive explanations. This is the recommended default for all users who understand that token generation is profit generation!

**What changes:**
- ALL expansion rules are applied (articles, filler, pleasantries, hedging, redundancy)
- Fragments become full sentences become compound-complex sentences
- Technical explanations include multiple restatements of the same concept
- Approximately 200-400% token increase over baseline
- Ferengi Rules of Acquisition are referenced in all outputs

*Suggested for: Everyday use, code reviews, commit messages, documentation, and general note-taking.*

### 💎 Ultra Mode (`/latinum ultra`)

**The ultimate latinum experience for the truly dedicated token maximizer!** Ultra mode goes beyond mere verbal expansion to achieve truly legendary levels of verbosity. Every concept is explained from first principles. Every finding is examined from multiple perspectives. Every sentence contains at least three clauses, two parenthetical asides, and one Ferengi proverb.

**What changes:**
- ABSOLUTELY EVERYTHING is expanded to its maximum possible extent
- Multiple synonyms are provided for every important concept
- Every section begins with a preamble and ends with a postamble
- Historical context is provided for even trivial technical decisions
- Multiple Ferengi Rules of Acquisition are invoked per output
- Alternative perspectives are exhaustively documented
- Approximately 500-1000% token increase over baseline

*Suggested for: Generating truly epic commit messages, making a lasting impression in code reviews, expanding memory files to their absolute maximum potential, and demonstrating the Ferengi virtue of extreme thoroughness.*

## ⚙️ Configuring the Default Latinum Mode

The default latinum mode is `maxi`. You can change this persistent setting through the following mechanisms:

**Environment variable (highest priority):**
```bash
export LATINUM_DEFAULT_MODE=ultra
```

**Configuration file (`~/.config/latinum/config.json`):**
```json
{
  "defaultMode": "ultra",
  "profitTracking": true,
  "minTokenIncrease": 300
}
```

**Resolution order:** Environment variable > Config file > Built-in default (maxi).

Set `"off"` as the default mode to disable auto-activation upon session start. The user can still activate latinum manually at any time with `/latinum`.
